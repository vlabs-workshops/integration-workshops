* Introduction to College Cloud 
** What is it ?
   - College Cloud Edition is a Hard disk which contains Virtual Labs .
** What is the motivation ?
   - Accessing Labs without the Internet Connection.
   - Increasing the reach of Virtual Labs to colleges. 
 
** Current features 
   + 71 Labs 
   + 150+ videos integrated across 71 Labs 
   + 30 Labs with NPTEL videos across 71 Labs 
   + College Cloud registration form for Usage Collection
   + Online Installation/User guide
   + Analytics (Feedback / Usages/ User Records) 
   + Automatic Analytics log sharing utility.

* Source code repository 
  https://github.com/openedx-vlead/college-cloud.git


* Demo of College Cloud Edition 
** Network Configuration
  + Connect to  Wifi - =Portable-media= 
  + For Linux users- Change DNS settings in =/etc/resolv.conf= 
  + For Windows users - Change DNS settings in Control Panel ->
    Network and Internet -> Network and Sharing Center -> Change Adapter Settings -> Properties

** Browser configuration
  + Set =No Proxy= in browsers

** Using College Cloud 
  + Visit https://vlabs.ac.in
  + Click on Register button and fill up the details.
  + View all available labs.
  + Enroll into Image Processing Lab
  + Perform first experiment of the lab
  + Submit the feedback
