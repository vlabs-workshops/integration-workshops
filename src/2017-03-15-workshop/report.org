* Objectives of the Workshop
  Objectives of the workshop were to -
  + Understand and follow the installation process and usage of the
    Virtual Labs College Cloud Edition.
  + Assist the Nodal Coordinators in conducting workshops using
    Virtual Labs College Cloud Edition.

* Summary of the Workshop
** First Day : March 15, 2017
*** Registration of participants ( 9:30 AM )       
    + Registration was carried out by VLEAD for all the participants.
    + Participants were requested to fill in their names and other
      details in the registration form.
    + Participants were given a handout.

*** Welcome Address by Professor Venkatesh Choppella ( 10:00 AM )
    A welcome message and introduction was given by Professor
    Choppella. He talked about the following -
    + How can we contribute to this to this project ? 
    + Praised the technology used by VLEAD used for web applications
    + Every one in this community must come on IRC channels for better
      and instant communication.
    + Thanked the VLEAD team and others who joined for this workshop
  
*** A talk on VLEAD process by Mr. Thirmal Ravula ( 10:20 AM )
    Mr. Thirmal deliverd a talk on VLEAD process and its components.
    + Displayed https://dev.vlabs.ac.in and process for integration,
      release management, milestones management.
    + Talked about the development process in detail.
    + How people can contribute and form a community.
    + Introduced IRC channel and other communication channels,
      engineers-forum, mailing list avilable for help within developer
      community.
    
    
*** Ice Breaking session with Mr. Priya Raman ( 10:40 AM )
    This was a small introduction session with an intension to know
    each other in the session.
    + Participants introduced themselves, and their working experience
      on Virtual Labs project.

*** Agenda of workshop by Ashay Maheshwari
    + Talked about the History of Virtual Labs projects.
    + Agenda of Day 1 and 2 was discussed where a brief introduction to
      each point in agenda was given.
    + Ashay ensured for a gradual progress and lot of possible
      interactions.

*** Talk by Mr. Vaibhav Khure on Portable version of Virtual Labs by COE Pune
    + presentation by Vaibhav kurhe on the common platform (portable
      version of virtual labs) that they are using and developed for
      adding all the labs and make them accessbile without any
      internet issues.
    + Questions on why multiple portals was raised by partipants.
    + Mr. Maruthi stated it as very good example of open source
      ideology (we can pull the different labs according to our needs
      and put it on our portal as per our needs). 
    + Displayed the Analytics system of vlabs.ac.in and talked about
      how ELK stack is used to capture analytics.

*** Introduction to College Cloud Edition by Ashay Maheshwari ( 10:45 AM )
    Ashay spoke about the following -
    + Introduced College Cloud Edition.
    + Mentioned the need to reach colleges facing internet issues and
      not able to utilise Virtual Labs.
    + Revealed the current feature of College Cloud Edition.
    + Briefly explained the Architecture Design diagram of College
      Cloud Edition.
