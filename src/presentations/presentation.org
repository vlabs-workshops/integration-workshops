* Introduction to College Cloud 
** What is it ?
   - College Cloud Edition is a Hard disk which contains Virtual Labs 
** What is the motivation ?
   - Labs accessible without the Internet Connection.
   - Increase the reach of Virtual Labs to colleges lacking internet connections.
 
** Current features 
   + Packaged with 73 Labs (Labs which are on Integration level 5 or 6)
   + College Cloud registration form
   + Installation/User guide
   + Feedback collection
   + Automatic log sharing utility for analytics
